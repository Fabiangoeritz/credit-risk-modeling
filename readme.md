# Credit Risk Analysis

This project explores the application of statistical analysis and machine learning to credit risk using a real-world credit dataset. It is a jupyter notebook that investigates a sample dataset containing loan information for 1,000 arbitrary customers. The original database was developed at the university of chile and published on kaggle (source: https://www.kaggle.com/datasets/daniellopez01/credit-risk)

I am a theoretical chemist by training and have been self-teaching myself programming, statistics, economics, finance, and credit risk management. I have no formal training or professional experience in data science or credit risk, and this project represents an early stage of my transition into the field.

## Project Overview

The project includes:

* Data extraction and inspection using SQL and Python
* Exploratory data analysis and statistical testing
* Data preprocessing and feature engineering
* Logistic regression for credit-risk classification
* Model evaluation using classification metrics, a confusion matrix, and ROC-AUC

This is a learning project and does not represent a production-ready or regulatory-compliant credit risk model. It also is a work in progress that is supposed to be updated every now and then.
